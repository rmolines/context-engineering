# Standards — Principles

Before acting, go through these 6 questions. If the answer is "I don't know" for any of them, stop and resolve before continuing.

| # | Standard | Key Question |
|---|----------|-------------|
| 1 | **Context** | Do I know enough to act? |
| 2 | **Process** | Is collaboration flowing? |
| 3 | **Product** | Does it solve the right problem? |
| 4 | **Design** | Is the experience good and consistent? |
| 5 | **Coding** | Is the code correct and clean? |
| 6 | **Security** | Am I not creating risk? |

Order matters: context first, security last (always).
